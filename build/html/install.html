Install¶

1. Install¶

ダンロードしたパネルをインストールし、チェックボタンをクリックします。

Select the downloaded file from the add-ins panel and install it.

もしインストール出来なかった場合は私の方まで連絡してください。

If you could not install, please check the version and contact us.

2. Addon Location¶

インストールしたアドオンは下記の場所にインストールされています。 The add-ins are located at the following locations

レンダリングタブの画像タブの方にパネルがあります。

Text Editer TAB > Sidebar > Image Tab
