How to use(Image Render)¶

イメージのレンダリング方法

1. Splist image editer¶

（レンダリング画面の分割）

このボタンを押す事によって素早くカメラ画面を左に作ることができます。

左ボタンは４回まで押せ、最高画面を四分割できます。

By pressing this button, you can quickly create a camera screen to the left.

The left button can be pressed up to four times to divide the screen into four sections.

2. Image Set¶

（イメージのセット方法）

画像のレンダリングに関して、主要なイメージ設定をここに集結しました。

The main image settings are gathered here with respect to image rendering.

背景の透明化や線の出力、 線とイメージの分離などの設定がこちらで可能になります。

You can set background transparency, line output, and line/image separation. You can also set the background transparency, line output, line and image separation, etc. here.
