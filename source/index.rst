Welcome to Render Slots Save's documentation!
=============================================
レンダースロットは画像レンダリングの簡易的な機能をイメージパネルに集結したアドオンです。

Render Slot is an add-on that gathers simple image rendering functions into an image panel.

いちいち他のパネルに移動変更しなくていいので、
レンダリングの作業時間の節約になります。

It saves time in rendering because you don't have to move and change to other panels every time you want to render.
This saves rendering time.
