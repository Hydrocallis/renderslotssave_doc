

How to use(Image Render)
========================
イメージのレンダリング方法

======================
1. Splist image editer
======================
（レンダリング画面の分割）


このボタンを押す事によって素早くカメラ画面を左に作ることができます。

左ボタンは４回まで押せ、最高画面を四分割できます。

By pressing this button, you can quickly create a camera screen to the left.

The left button can be pressed up to four times to divide the screen into four sections.

|splist_image_editer|


.. image:: /img/howtouse/camera_set.gif
   :scale: 60%

You can set background transparency, line output, and line/image separation.
